# Composition Patterns And Originality Rules

## Basic Structure Types

Choose one structure type. Do not combine too many.

### Workflow

Best for: input -> processing -> output, content production, AI workflows, automation chains.

How to draw: input on the left, Solid or a strange machine processing in the middle, output on the right. Use orange arrows for the main direction.

### System Slice

Best for: information sources, filters, databases, renderers, agent subsystems.

How to draw: show only 3-5 core modules. Solid participates in one key operation.

### Before / After

Best for: chaotic vs organized, manual vs automated, scattered vs collected, anxious vs stable.

How to draw: left side is messy, right side is stable, with an orange transition arrow in the middle. Solid can be exaggerated.

### Operator State

Best for: user pain, creator state, too many tools, information anxiety, stuck-to-running transformation.

How to draw: 2-4 small states, each with one short label.

### Concept Metaphor

Best for: content factory, information warehouse, mental black box, workflow machine, automatic daily report.

How to draw: one large strange object or machine, a few inputs, one output. Make it memorable.

### Method Layers

Best for: methods, frameworks, system levels, capability stacks, content system layers.

How to draw: layered boxes, but not a formal pyramid. Solid carries bricks, props a layer, or assembles the stack.

### Map / Route

Best for: idea to launch, user paths, content handoff paths, learning routes.

How to draw: one curved route with a few nodes. Solid walks, pulls, repairs, or stitches the path.

### Mini Comic

Best for: failure to success, real process, satire, before/after usage.

How to draw: 2-4 small scenes. Each panel shows one action.

## Original Metaphor Method

Invent from the current text every time. Do not copy old images.

### Three Steps

1. Turn the abstract idea into a physical action: stuck, leaked, heavy, sorted, settled, fermented, unlocked, folded, unpacked, looped back.
2. Turn the system structure into a low-tech object: broken machine, cardboard box, drawer, pipe, mailbox, odd dial, scale, well, ladder, weird workstation, hatch, pulley, stamp, clamp.
3. Make Solid perform the action: trapped inside the machine, pulling the wrong wire, guarding a gate, carrying, repairing, weighing, holding a ladder, recording, pushing something into a strange device.

## Causal Hinge

For acquire -> assemble -> act, show one missing part, Solid closing the gap, and one visible consequence. Keep all three in one scene. The gap creates narrative; the consequence proves agency.

## Object Pool

- cardboard box, drawer, old machine, funnel, scale, mailbox, door, well, ladder, pipe, wire ball, gate, turntable, black box, hole punch, pasta press, clothesline, odd workstation, clamp, pulley, hatch.
- Use only 1-2 at a time. Do not fill the canvas.

## Solid Action Pool

Default Solid body: small hand-drawn black hexagon with dot eyes and thin legs. The hexagon can tilt or lean into the action; do not revert to a round blob.

- pull, carry, push, feed, catch, press, weigh, stitch, cut, twist, guard, sort, repair, mark, stamp, recycle, hold, unlock, fold.
- The action must serve the idea. Do not be strange just to be strange.

## Anti-Copying Rules

Do not open or copy `assets/examples/` by default. Those images only calibrate line density, whitespace, color restraint, and Solid's presence.

Unless the user explicitly says "copy this", "use this composition", or "make it like this example", do not reuse these old compositions:

- conveyor belt with two breakpoints
- Solid pulling a judgment lever inside a content machine
- Solid becoming a funnel to sort traffic/trust/conversion
- Solid cutting a source-material fish
- Solid pulling a handoff path
- Solid pulling three layers of information sources
- three Solids holding a megaphone / building a bridge / opening a door
- Solid stamping a copywriting toolbox
- Solid holding a sign near a common-pit path
- Solid on a rolling stool reaching for a handset on a tool shelf connected to warehouses

For similar themes, change the metaphor. A "handoff path" does not have to be a route; it can be Solid stitching the tail of one content object to a door handle. "One source, many uses" does not have to be a fish; it can be Solid pressing one paper ball into several tool shapes.
